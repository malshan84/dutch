    N빵 계산기 사용 방법

    N빵 계산기를 사용하여 더치페이와 비용 정산을 쉽고 정확하게 할 수 있습니다. 다음은 단계별 사용 방법입니다:

    기본 사용법

    1. 항목 입력: '항목 이름'에 계산할 항목(예: 식사, 영화 등)을 입력합니다.
    2. 가격 입력: '가격'에 해당 항목의 총 비용을 입력합니다.
    3. 인원 입력: '부담할 사람'에 비용을 나눌 사람들의 이름을 쉼표로 구분하여 입력합니다.
    4. 항목 추가: '항목 추가' 버튼을 클릭하여 입력한 항목을 목록에 추가합니다.
    5. 추가 항목: 필요한 만큼 2-4단계를 반복하여 모든 항목을 추가합니다.
    6. 계산: 모든 항목을 입력한 후 '계산하기' 버튼을 클릭합니다.
    7. 결과 확인: 각 개인별로 부담해야 할 금액을 확인합니다.
    8. 결과 공유: '결과 복사' 버튼을 클릭하여 계산 결과를 클립보드에 복사하고 필요한 곳에 붙여넣어 공유합니다.

    항목 관리하기

    N빵 계산기에서는 항목을 쉽게 추가, 수정, 삭제할 수 있습니다. 다음은 각 작업의 상세한 방법입니다:

    항목 추가하기

    1. 항목 이름, 가격, 부담할 사람을 입력합니다.
    2. '항목 추가' 버튼을 클릭합니다.
    3. 추가된 항목이 목록에 나타납니다.

    항목 수정하기

    1. 수정하고 싶은 항목 옆의 '수정' 버튼을 클릭합니다.
    2. 항목 정보가 입력 필드에 자동으로 채워집니다.
    3. 원하는 정보를 수정합니다.
    4. '항목 수정' 버튼을 클릭하여 변경사항을 저장합니다.

    항목 삭제하기

    1. 삭제하고 싶은 항목 옆의 '삭제' 버튼을 클릭합니다.
    2. 항목이 즉시 목록에서 제거됩니다.

    주의: 항목을 삭제하면 복구할 수 없으니 신중히 선택하세요.
